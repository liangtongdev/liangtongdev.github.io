MQTT

MQTT是工作在TCP/IP协议族上的发布/订阅范式消息的协议。
跟HTTP协议一样，不能直接拿来使用，需要找实现这个协议的库或者服务器来运行。
本文介绍服务器apache-apollo的配置。
